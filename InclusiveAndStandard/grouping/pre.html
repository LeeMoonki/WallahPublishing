pre

<pre> 요소는 preformatted text 블럭을 나타낸다. preformatted text에서의 구조는 HTML 요소가 아닌 활자체의 작성법에 의해 나타난다.

<pre> 요소는 다음과 같은 경우에 사용될 수 있다.

  • 비어있는 줄, 점으로 구분된 리스트 등의 문장들
  • 작성법이 정해져 있는 컴퓨터 코드
  • ASCII가 포함된 문장

This is the Panel constructor:

    
function Panel(element, canClose, closeHandler) {
  this.element = element;
  this.canClose = canClose;
  this.closeHandler = function () { if (closeHandler) closeHandler() };
}
