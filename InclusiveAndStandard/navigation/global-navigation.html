Current Page

navigation은 <nav> 태그 내부에 <ul> 태그를 사용해 하위호환을 고려할 수 있고 CSS가 동작하지 않는 상황에도 적절한 navigation을 제공할 수 있다.

색상으로 현재 페이지를 구분하게 되면 색맹을 가진 사용자에게 접근성이 떨어지는 navigation을 제공하게 된다. 따라서 색상보단 밑줄이나 다른 시각적 요소를 제공하는게 더 좋다.

청각적으로 현재 페이지의 정보를 전달하기 위해 'aria-describedby'와 hidden된 요소를 사용하거나 링크 내부에 visually-hidden된 설명 요소를 추가합니다. (aria-describedby를 사용하면 잠시 쉬었다가 설명을 읽어줍니다.)

청각적으로 현재 페이지의 정보를 전달하는 다른 방법은 href="#main"입니다. 이렇게 하면 'current page'라는 것 없이 현재 페이지라고 스크린 리더가 읽어줍니다. 또한 링크를 클릭했을 때 페이지를 새로고침하지 않고 현재 페이지를 유지합니다.
